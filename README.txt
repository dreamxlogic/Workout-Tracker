Gym Buddy Local — V1

How to use:
1. Keep the entire gym-app folder together.
2. Open index.html in a browser.
3. Your workout data saves locally in the browser on that device.
4. Use Settings > Backup & Restore > Create Backup to save a JSON backup file.
5. Use Settings > Backup & Restore > Import Backup to restore a backup file.

Important notes:
- This app is local HTML/CSS/JavaScript. No backend, login, cloud, API, or AI model is used.
- If you clear browser data, local workout data may be deleted unless you exported a backup.
- The image folder includes generated SVG placeholder assets and mask-ready files for theme accent recoloring.
- This app is not affiliated with Planet Fitness. The equipment library is a local, user-maintained Planet Fitness-style gym equipment list.

Theme behavior:
- Theme colors change buttons, active nav, chips, tabs, selected states, machine accent masks, avatar clothing masks, and muscle map overlays.
- Theme colors do not change skin tone, hair, neutral machine frames, semantic warning/error colors, or badge medal materials.
